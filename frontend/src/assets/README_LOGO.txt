# Logo Assets

This directory contains logo and brand assets for the Zora Music application.

## Current Logo
The logo is currently generated using Lucide React icons in the Logo.tsx component.
No custom logo file is required.

## Future Assets
If you want to add custom logo files, place them here:
- logo.svg - Main SVG logo
- logo.png - PNG version for various uses
- favicon.ico - Browser favicon
- logo-light.svg - Light theme variant
- logo-dark.svg - Dark theme variant

## Guidelines
- Maintain consistent branding across the application
- Use SVG format when possible for scalability
- Keep file sizes optimized for web performance
